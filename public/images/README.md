# Imagini de fundal

Pune aici fișierele descărcate de pe Unsplash (sau altă sursă cu licență liberă comercială). Nume așteptate:

- `testimonials-bg.jpg` — fundal secțiune Testimonials (meadow/flori, culori calde)
- `cta-bg.jpg` — fundal secțiune CTA final (munte/lumină de seară, sau abstract)
- `hero-texture.jpg` — opțional, textură subtilă în spatele hero-ului

Dacă un fișier lipsește, secțiunea respectivă cade automat pe gradient-ul CSS existent, deci site-ul nu se rupe niciodată din lipsa unei imagini.
